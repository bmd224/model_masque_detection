# model_masque_detection
Ce projet de détection de masque facial utilise un CNN TensorFlow entraîné sur le dataset Kaggle Face Mask Detection comprenant des images avec masque et sans masque.
Le script model_masque_detection.py télécharge le dataset, prépare les images, entraîne un modèle TensorFlow/Keras, affiche les courbes d’apprentissage, évalue la performance et permet de prédire simplement sur une nouvelle image fournie par l’utilisateur
